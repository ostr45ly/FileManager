1. Программа предлагает для работы с файлами меню из 6 пунктов.
   Пользователю  необходимо ввести число соответствующие номеру пункта меню.

2. Перед началом работы необходимо ввести путь по предлагаемому шаблону С:\<папка>\
   Программа проверяет существует ли  каталог , если отсутствует создает новый каталог.
   Если путь не соответствует шаблону - окончание работы. 

   Применяется  класс File , Pattern

3. Пользователю предлагается строка меню : Create file.
   С консоли пользователь вводит любое имя создаваемого файла, например: QA10.TXT
   Новый файл заполняется данными:
   Gruppa QA_10 labwork #1 (число повторений 5 раз), для дальнейшей работы по замене и поиску слов.
   При успешном создании файла на консоли появиться сообщение - Файл создан.
   При неуспешном создании файла на консоли появиться сообщение - Файл не создан. 
   Применяется  класс: Scanner,FileWriter,BufferedWriter

4. При успешном создании файла пользователю предлагается строка меню: Rename file.
    С консоли пользователь вводит имя заменяемого файла, например: QA10.TXT
    С консоли пользователь вводит имя файла для переименования, например: TEST.TXT
    При успешном переименовании  файла на консоли появиться сообщение - Файл переименован.

    Применяется  класс: Scanner,File.

5.  Cтрока меню: Find word.
     При успешном переименованиии файла пользователю предлагается   содержимое переименованного файла:
     Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1

    С консоли пользователь вводит слово для поиска например: QA 
    В случае успешного поиска на консоли появиться сообщение - слово найдено.
    Применяется  класс: Scanner,BufferedReader,FileReader

6.  Cтрока меню: Replace word.
     При успешном переименованиии файла пользователю предлагается   содержимое переименованного файла:    
     Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1 Gruppa QA_10 labwork #1
   
    С консоли пользователь вводит слово поиска в заменяемой строке например: QA
    С консоли пользователь вводит слово для замены например: QAuto

    В случае успешного поиска на консоли появиться сообщение - замена в количестве = 5.
    Применяется  класс: Scanner,BufferedReader,FileReader

7.  Cтрока меню:Delete file
    С консоли пользователь вводит  имя файла.
    В случае успешной операции на консоли появиться сообщение -Файл удален: TEST.TXT
    Применяется  класс: Scanner,File.
